Installing
==

    npm install
    npm install --only=dev

Running
==

    npm run build	# npm run build:prod for production
    npm start

In the .env file add
==
    MC_INSTANCE_ID = {region/instance}
    MC_LIST_ID = {list id}
    MC_API_KEY = {api key}
    MG_PUBLIC_KEY = {mailgun public api key}
